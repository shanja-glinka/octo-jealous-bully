## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
$ cd apps/ttl-service
$ pip install -r requirements.txt
```

## Running the apps

```bash
# text to speech service
$ python tts_service.py

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
